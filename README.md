## 🌐 Overview

Visadorm P2P is a **non-custodial peer-to-peer platform** that enables users to transact directly using **smart contract escrow**.

Built on **Base (Ethereum Layer 2)** and powered by **USDC**, the platform provides a secure and efficient way for individuals to exchange value without intermediaries.

---

## ⚙️ How It Works

1. A transaction is initiated between two users  
2. Funds are locked in a smart contract escrow  
3. Conditions are agreed between participants  
4. Funds are released automatically once conditions are met  

> No intermediary custody. No centralized control.

---

## 🔑 Core Principles

- **Non-Custodial** — Users retain control of their funds at all times  
- **Peer-to-Peer** — Direct interaction between participants  
- **On-Chain Security** — Escrow enforced by smart contracts  
- **Transparency** — Verifiable transactions on blockchain

## 🧱 Tech Stack

- **Backend:** Laravel (PHP)  
- **Smart Contracts:** Solidity  
- **Blockchain:** Base (Ethereum L2)  
- **Stablecoin:** USDC  

---

## 💵 About USDC

USDC is a fully reserved digital dollar stablecoin issued by Circle, designed to maintain a 1:1 peg with the US dollar. It enables fast, transparent, and reliable on-chain payments.

---

## 🔒 Non-Custodial Notice

This platform does **not** custody user funds.

All transactions are executed directly between participants through smart contracts.  
The platform provides infrastructure only.

---

## ⚠️ Disclaimer

This software provides peer-to-peer infrastructure and does not operate as a financial intermediary, broker, exchange, or money transmitter.
